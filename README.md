# mpq-builder

CLI tool and TypeScript library to read, create and manage MPQ (Mo'PaQ) archives — the archive format used by World of Warcraft 3.3.5a (WotLK).

Built on [StormLib](https://github.com/ladislav-zezula/StormLib) via native bindings [@jamiephan/stormlib](https://www.npmjs.com/package/@jamiephan/stormlib).

## Installation

```bash
npm install
npm run build
```

## CLI

```bash
npm run mpq -- <command> [options]
```

### Commands

#### `list` (alias: `ls`)

List files in an MPQ archive.

```bash
npm run mpq -- list patch-3.MPQ
npm run mpq -- list -l patch-3.MPQ    # with file sizes
```

#### `info`

Display archive information (size, file count, max capacity).

```bash
npm run mpq -- info patch-3.MPQ
```

#### `extract` (alias: `x`)

Extract one or all files from an archive.

```bash
# Extract all files to a directory
npm run mpq -- extract patch-3.MPQ -o ./output

# Extract a specific file
npm run mpq -- extract patch-3.MPQ "DBFilesClient\Spell.dbc" -o ./output
```

#### `create` (alias: `c`)

Create a new MPQ archive from files or directories.

```bash
# From a directory (structure preserved)
npm run mpq -- create my-archive.MPQ ./my-folder

# From multiple sources with a prefix
npm run mpq -- create my-archive.MPQ file1.dbc file2.dbc -p DBFilesClient

# With a custom max file count
npm run mpq -- create my-archive.MPQ ./data -n 8192
```

#### `add` (alias: `a`)

Add files or directories to an existing archive.

```bash
# Add a single file
npm run mpq -- add patch-3.MPQ my-file.dbc -p DBFilesClient

# Add a whole directory
npm run mpq -- add patch-3.MPQ ./my-files

# Replace existing files
npm run mpq -- add patch-3.MPQ ./my-files -r
```

#### `remove` (alias: `rm`)

Remove a file from an archive.

```bash
npm run mpq -- remove patch-3.MPQ "DBFilesClient\Spell.dbc"
```

#### `rename` (alias: `mv`)

Rename a file inside an archive.

```bash
npm run mpq -- rename patch-3.MPQ "old\path.dbc" "new\path.dbc"
```

#### `patch`

Create a WoW 3.3.5a-ready patch MPQ from a directory. The source directory should mirror the game's internal structure (`DBFilesClient/`, `Interface/`, etc.).

```bash
# Prepare your files
mkdir -p patch-data/DBFilesClient
cp Spell.dbc patch-data/DBFilesClient/

# Generate the patch
npm run mpq -- patch ./patch-data -o patch-4.MPQ

# Deploy: copy patch-4.MPQ into <WoW>/Data/
```

The patch number (`patch-X.MPQ`) must be higher than existing patches in `Data/` so the client loads it with priority.

## Programmatic API

The `MpqManager` class can be imported into any TypeScript/JavaScript project.

```typescript
import { MpqManager } from "mpq-builder";

const mpq = new MpqManager();

// Read an existing archive
mpq.open("patch-3.MPQ", true); // true = read-only
const files = mpq.listFiles();       // { name, size }[]
const info = mpq.getInfo();          // { path, fileCount, maxFileCount, files }
const exists = mpq.hasFile("DBFilesClient\\Spell.dbc");
const buffer = mpq.readFile("DBFilesClient\\Spell.dbc");
mpq.close();

// Extract files
mpq.open("patch-3.MPQ", true);
mpq.extractFile("DBFilesClient\\Spell.dbc", "./output/Spell.dbc");
mpq.extractAll("./output");
mpq.close();

// Create a new archive
mpq.create("patch-custom.MPQ", {
  maxFileCount: 4096,
  listfile: true,
  attributes: true,
});
mpq.addFile("./Spell.dbc", { archivePath: "DBFilesClient\\Spell.dbc" });
mpq.addDirectory("./my-folder");
mpq.compact();
mpq.close();

// Modify an existing archive
mpq.open("patch-custom.MPQ");
mpq.addFile("./new-file.dbc", {
  archivePath: "DBFilesClient\\NewFile.dbc",
  replace: true,
  compress: true,
});
mpq.removeFile("DBFilesClient\\OldFile.dbc");
mpq.renameFile("old\\path.txt", "new\\path.txt");
mpq.compact();
mpq.close();
```

## Technical notes

- MPQ v1 format (compatible with WoW 3.3.5a, max 4 GB per archive)
- ZLIB compression by default
- Internal paths use backslashes (`DBFilesClient\Spell.dbc`) as per MPQ convention
- Internal files `(listfile)`, `(attributes)` and `(signature)` are automatically skipped during full extraction
